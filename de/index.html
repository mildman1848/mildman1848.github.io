Dieses Repository stellt eigene Kodi-Add-ons sowie ausgewahlte externe Quellen bereit. Englisch ist die Standardsprache dieser Seite, zusatzlich steht eine deutsche Version zur Verfugung.

So installierst du das Repository in Kodi

  1. Offne in Kodi den Dateimanager und wahle Quelle hinzufugen.
  2. Trage https://mildman1848.github.io/ als Pfad ein.
  3. Vergib einen Namen, zum Beispiel mildman1848 Repo.
  4. Wechsle zu Add-ons und wahle Aus ZIP-Datei installieren.
  5. Wahle die Quelle und installiere repository.mildman1848-1.0.7.zip.

Nach der Installation nutzt Kodi den Repository-Feed unter /repo/addons.xml.
