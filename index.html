This repository provides our own Kodi add-ons and selected maintained packages. It includes official projects like Kodi4Seniors and maintained forks like TMDb Helper for headless Kodi use cases.

Install In Kodi

  1. Open File Manager in Kodi and choose Add source.
  2. Enter https://mildman1848.github.io/repo/repository.mildman1848/ as source path.
  3. Name it, e.g. mildman1848 Repo.
  4. Open Add-ons and choose Install from zip file.
  5. Install repository.mildman1848-1.0.9.zip.

Kodi File Manager does not reliably list ZIP files from the styled overview pages at / or /repo/. Use the direct repository ZIP folder above instead.
