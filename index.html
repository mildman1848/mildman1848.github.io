This repository provides own Kodi add-ons together with a curated set of external sources. English is the default language for this site; a German version is available as well.

How to install this repository in Kodi

  1. Open File Manager in Kodi and choose Add source.
  2. Enter https://mildman1848.github.io/ as the source path.
  3. Name the source, for example mildman1848 Repo.
  4. Open Add-ons and choose Install from zip file.
  5. Select the source and install repository.mildman1848-1.0.7.zip.

After installation, Kodi will use the repository feed at /repo/addons.xml.
