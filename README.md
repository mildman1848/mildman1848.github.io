# mildman1848.github.io

Deutsch: [README.DE.md](README.DE.md)

Static GitHub Pages site for the Kodi repository feed maintained under `mildman1848.github.io`.

## Purpose

- Publishes the `repository.mildman1848` feed.
- Mirrors selected repository ZIP files and metadata.
- Provides English and German package overview pages.

## Main Areas

- `index.html`: site landing page
- `de/`: German pages
- `repo/`: repository feed, package folders, and ZIP downloads
- `assets/`: shared site styles and scripts
